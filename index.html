Contatos:

Certificados

  • EF Core: Relacionando Entidades

    Este curso fornece um mergulho profundo nos estudos de relacionamentos entre entidades no contexto do .NET e Entity Framework Core, abrangendo desde relações um-para-um (1:1) até um-para-muitos (1:n) e muitos-para-muitos (n:n). Foram relizadas atividades práticas e teóricas para aplicação de com conhecimentos sólidos e habilidades práticas na modelagem e implementação de relacionamentos de banco de dados.

  • Fundamentos do Event-Driven

    O curso abordou Event-Driven Architecture (Arquitetura Orientada a Eventos), incluindo demonstrações de mapeamento, conceitos-chave como eventos, políticas de execução, integração com sistemas externos, e fundamentos teóricos de mensagens, eventos e comandos. Recomendou-se também explorar CQRS para uma compreensão mais aprofundada. O professor Andre Baltieri apresentou um projeto prático durante o curso.

  • Segurança em APIs ASP.NET

    Este curso oferece um estudo aprofundado sobre a implementação de segurança em APIs ASP.NET por meio do uso do JWT (JSON Web Tokens) e da autenticação Bearer. As aulas abordam tanto aspectos teóricos quanto práticos, garantindo uma compreensão completa de como essas tecnologias funcionam e como podem ser efetivamente aplicadas para proteger suas aplicações web.

  • Fundamentos do SQL Server

    Este curso aborda de forma prática e interativa os conceitos essenciais do SQL, utilizando o SQL Server como ferramenta de aprendizado. O instrutor fornece um banco de dados dedicado para que os alunos possam aplicar e aprimorar suas habilidades em SQL, realizando exercícios práticos. As queries e técnicas são ensinadas passo a passo durante as aulas, facilitando o entendimento e a aplicação dos conceitos aprendidos.

  • React + TypeScript

    Nesta aula, exploramos os conceitos fundamentais do React, incluindo componentização, props e state, com a integração poderosa do TypeScript. O curso se aprofunda em um projeto prático, onde os conceitos de hooks, como useState e useEffect, são detalhadamente abordados. Além disso, enfatizamos a aplicação de boas práticas no desenvolvimento, garantindo uma base sólida para a construção de aplicações React eficientes e escaláveis.
